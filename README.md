# Yazan Ghayad

Full-stack developer, researcher, and founder of OptiTech Sverige AB. Based in Sweden.

I build software products that solve real business problems, with a focus on AI-powered platforms, customer communication tools, and workforce management systems.

## Current Projects

**[SWEO](https://github.com/yazanghayad/SWEO)** -- AI-powered customer support platform with multi-channel inbox, knowledge base management, conversation analytics, and an embeddable chat widget. Built with Next.js, React, MongoDB, OpenAI, and Stripe.

**[Prakto](https://github.com/yazanghayad/prakto)** -- Internship and career management platform connecting students with companies. Features application tracking, internship journals, portfolio building, and AI-assisted tools. Built with Next.js, React, Appwrite, and TanStack.

## Tech
